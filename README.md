# Idle Face

A gamified NFT PFP mini app built on Farcaster using the Neynar SDK.

## Overview

Idle Face is a collection of 555 unique robot avatar NFTs on Base. Users mint NFTs for $1.50 USDC each, and 50% of revenue goes to a prize pool split between top collectors.

## Features

- NFT Minting: Mint unique Idle Face robot PFPs for $1.50 USDC
- Leaderboard: Real-time rankings based on NFT holdings
- Prize Pool: Top 5 holders + 10 random holders split 50% of revenue
- Social Gates: Follow, like, recast requirements before minting
- Profile Gallery: View your collection with up to 20 NFTs displayed

## Tech Stack

- Framework: Next.js 16 with React 19
- Blockchain: Base (ERC-721 NFT contract)
- Payments: USDC on Base
- Social: Farcaster via Neynar SDK
- Database: Neon PostgreSQL with Drizzle ORM
- Styling: Tailwind CSS + Neynar UI components

## Smart Contract

- Address: 0x8F8Bdfd873F82179f0c75124BFeCCcee07Aa9845
- Chain: Base Mainnet
- Standard: ERC-721 with ERC-721Enumerable

## Gates Before Minting

1. Follow @web3focus on Farcaster
2. Add the mini app
3. Like & recast the launch cast

## Prize Distribution

- 50% of mint revenue goes to prize pool
- Top 5 holders by NFT count
- 10 random holders selected
- Distributed after mint sells out
